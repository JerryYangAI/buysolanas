## Saga Phone

**Saga** is a crypto-native Android smartphone created by Solana Mobile. It features a built-in secure element for private key storage, a Solana-focused dApp Store, and deep integration with the Solana ecosystem for seamless mobile crypto experiences.

### How It Works

The Saga phone includes a Seed Vault, a secure hardware enclave that stores private keys separately from the main operating system. This allows users to sign transactions securely without exposing their keys. The Solana dApp Store provides a curated marketplace of mobile-optimized decentralized applications.

### Why It Matters

Saga represents the vision of bringing crypto to mobile users in a secure, user-friendly way. By integrating hardware-level security with a purpose-built dApp ecosystem, it aims to make blockchain interactions as natural as using any other mobile app while maintaining self-custody of assets.
