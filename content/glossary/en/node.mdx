## Node

A **node** is a computer running blockchain software that participates in the network by maintaining a copy of the blockchain, validating transactions, and relaying data to other nodes. Nodes are the infrastructure that keeps a blockchain decentralized and operational.

### How It Works

Nodes connect to each other in a peer-to-peer network. They receive new transactions, verify them against the protocol rules, and propagate valid ones. Different types of nodes serve different purposes: validator nodes produce blocks, full nodes maintain the complete blockchain history, and RPC nodes provide API access for applications.

### Why It Matters

The number and geographic distribution of nodes determines a blockchain's decentralization and resilience. More nodes mean more copies of the data and more entities verifying transactions, making the network harder to attack or censor. Anyone can run a Solana node to independently verify the chain.
