## Switchboard

**Switchboard** is a permissionless oracle network that allows developers to bring any off-chain data on-chain. Originally built for Solana, it supports customizable data feeds that can pull information from any API, making it highly versatile for DeFi and other blockchain applications.

### How It Works

Developers create data feeds by specifying which data sources to query and how to aggregate results. A network of oracle operators then fetches and verifies this data, publishing it on-chain. Switchboard also supports verifiable random functions (VRF) for applications that need provably fair randomness, such as gaming and NFT minting.

### Why It Matters

While Pyth focuses on financial price data, Switchboard offers broader oracle functionality. Any data available via an API can be brought on-chain, enabling use cases like sports betting results, weather data, and social media metrics to power smart contracts on Solana.
