## Firedancer

**Firedancer** is a new Solana validator client being developed by Jump Crypto from scratch in C/C++. It aims to dramatically increase Solana's throughput capacity and improve network resilience by providing a second, independent implementation of the Solana protocol.

### How It Works

Firedancer reimagines the validator architecture with optimizations at every level, from networking to transaction processing. It uses novel techniques for parallel transaction execution and memory management. Having a second validator client also means a bug in one implementation is less likely to take down the entire network.

### Why It Matters

Client diversity is crucial for blockchain resilience. If all validators run the same software, a single bug could halt the network. Firedancer provides redundancy while also pushing performance boundaries, with early benchmarks suggesting potential throughput of over 1 million transactions per second.
