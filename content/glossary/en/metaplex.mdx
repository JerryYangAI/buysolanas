## Metaplex

**Metaplex** is the foundational NFT infrastructure protocol on Solana. It provides the token metadata standard used by virtually all Solana NFTs, along with tools for minting, selling, and managing digital assets including candy machines for launches and auction houses for trading.

### How It Works

Metaplex defines how NFT metadata is structured on Solana, including attributes like name, image, traits, and royalty settings. Its Candy Machine program automates NFT collection launches with features like allowlists, payment options, and reveal mechanics. The protocol also supports programmable NFTs with enforced royalties.

### Why It Matters

Metaplex is the backbone of Solana's NFT ecosystem. By providing standardized infrastructure, it ensures interoperability between marketplaces, wallets, and tools. Its compressed NFT support and programmable royalty enforcement have set new standards for the broader NFT industry.
