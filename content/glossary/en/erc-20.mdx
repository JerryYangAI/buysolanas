## ERC-20

**ERC-20** is the technical standard for fungible tokens on the Ethereum blockchain. It defines a common set of rules that all Ethereum tokens must follow, ensuring they work with wallets, exchanges, and DeFi protocols.

### How It Works

The ERC-20 standard specifies functions like transfer, balanceOf, approve, and transferFrom. Any token implementing these functions is ERC-20 compliant and automatically compatible with the entire Ethereum ecosystem. This standardization was a key driver of the ICO boom and DeFi growth on Ethereum.

### Why It Matters

Understanding ERC-20 helps Solana users appreciate the equivalent SPL Token standard. While the implementation differs, both serve the same purpose: creating a universal interface for fungible tokens. Cross-chain bridges translate between these standards when moving tokens between Ethereum and Solana.
