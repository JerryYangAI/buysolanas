## dApp (Decentralized Application)

A **dApp** is an application that runs on a decentralized blockchain network rather than centralized servers. dApps use smart contracts for their backend logic and typically interact with users through a web interface connected to a crypto wallet.

### How It Works

dApps combine traditional frontend technology (websites, mobile apps) with blockchain-based backend logic (smart contracts). When a user performs an action, the frontend sends a transaction to the smart contract on the blockchain. The transaction is processed by validators, and the state change is recorded permanently on-chain.

### Why It Matters

dApps offer unique advantages over traditional applications: censorship resistance, transparency, user ownership of data and assets, and global accessibility. The Solana ecosystem hosts hundreds of dApps spanning DeFi, NFTs, gaming, social media, and infrastructure.
