## Metadata

**Metadata** in the crypto context refers to the descriptive information associated with an NFT or token. For NFTs, this includes the name, description, image URL, attributes (traits), and other properties that define what the NFT represents. Metadata is what gives an NFT its visual identity and uniqueness.

### How It Works

When an NFT is minted on Solana, its metadata is typically stored using the Metaplex standard. The on-chain token points to a metadata URI (usually stored on Arweave or IPFS for permanence) that contains a JSON file describing the NFT's properties. This JSON includes the image link, name, description, collection info, and an attributes array that defines traits like rarity characteristics.

### Why It Matters

Metadata is what makes each NFT distinct and valuable. Without metadata, an NFT would just be a meaningless token. The quality and permanence of metadata storage matters — NFTs with metadata stored on decentralized storage (like Arweave) are considered more reliable than those hosted on regular web servers that could go offline. When buying NFTs, checking the metadata can reveal important details about rarity and authenticity.
