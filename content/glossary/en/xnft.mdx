## xNFT

**xNFTs** (executable NFTs) are a concept pioneered by the Backpack wallet on Solana. They represent full-fledged applications tokenized as NFTs, allowing users to own, trade, and run decentralized apps directly within their wallet.

### How It Works

An xNFT contains or references an entire application bundle. When a user owns an xNFT, they can launch the application directly from their wallet interface without navigating to a separate website. The application runs in a sandboxed environment within the wallet, with direct access to the user's accounts and tokens.

### Why It Matters

xNFTs reimagine how users interact with decentralized applications. Instead of visiting websites that could be phished or spoofed, users run verified applications directly in their wallet. This model also allows developers to distribute and monetize applications as tradeable tokens.
