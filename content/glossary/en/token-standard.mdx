## Token Standard

A **token standard** is a set of rules and interfaces that define how tokens are created, transferred, and managed on a blockchain. Standards ensure that all tokens following them are compatible with wallets, DEXs, and other applications.

### How It Works

Token standards specify the functions a token contract must implement, such as transferring tokens, checking balances, and approving spending. On Ethereum, ERC-20 is the fungible token standard. On Solana, the SPL Token program serves as the standard for all fungible tokens, with Token Extensions (Token-2022) adding advanced features.

### Why It Matters

Without token standards, every token would need custom integration with every application. Standards create a universal interface that enables any compliant token to work with any compliant wallet, DEX, or DeFi protocol automatically.
