## Token Burn

A **token burn** is the process of permanently removing tokens from circulation by sending them to a dead wallet address that no one can access. This reduces the total supply, potentially increasing the value of remaining tokens through scarcity.

### How It Works

Burns are executed by sending tokens to a verifiable burn address (an address with no known private key). Some protocols burn tokens automatically as part of their fee mechanism — for example, burning a percentage of every transaction fee. Others conduct scheduled burns or buyback-and-burn programs funded by protocol revenue.

### Why It Matters

Token burns create deflationary pressure on the supply, which can support price appreciation over time. They signal that a project is committed to creating value for holders. However, burns alone do not guarantee value — the underlying utility and demand for the token are equally important.
