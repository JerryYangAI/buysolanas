## 元数据（Metadata）

加密领域中的**元数据**是指与 NFT 或代币相关联的描述性信息。对于 NFT，这包括名称、描述、图像 URL、属性（特征）以及定义 NFT 所代表内容的其他属性。元数据赋予了 NFT 视觉身份和独特性。

### 工作原理

当 NFT 在 Solana 上铸造时，其元数据通常使用 Metaplex 标准存储。链上代币指向一个元数据 URI（通常存储在 Arweave 或 IPFS 上以确保持久性），其中包含描述 NFT 属性的 JSON 文件。这个 JSON 包括图像链接、名称、描述、集合信息和定义稀有度特征等属性的属性数组。

### 为什么重要

元数据是使每个 NFT 独特和有价值的关键。没有元数据，NFT 只是一个无意义的代币。元数据存储的质量和持久性很重要——存储在去中心化存储（如 Arweave）上的 NFT 元数据被认为比托管在可能下线的普通网络服务器上的更可靠。在购买 NFT 时，检查元数据可以揭示有关稀有度和真实性的重要细节。
