## SPL Token（SPL 代币）

**SPL Token** 是 Solana 的标准代币程序，相当于以太坊的 ERC-20。它定义了如何在 Solana 区块链上创建、转移和管理同质化代币，确保整个生态系统的通用兼容性。

### 工作原理

SPL Token 程序是任何开发者都可以使用来创建代币的共享链上程序。每个代币都有一个铸币账户（定义供应量和小数位）和每个持有人的独立代币账户。该程序处理所有代币操作：铸造、转移、销毁和委托。较新的 Token-2022 程序扩展了额外功能。

### 为什么重要

SPL Token 是 Solana 代币经济的基础。Solana 上的每个同质化代币——从 SOL 包装到稳定币到模因币——都使用 SPL Token 程序。理解它帮助用户理解代币账户创建、租金豁免押金，以及代币如何在 Solana 生态系统中流动。
