## Metaplex（Metaplex 协议）

**Metaplex** 是 Solana 上的基础 NFT 基础设施协议。它提供了几乎所有 Solana NFT 使用的代币元数据标准，以及用于铸造、销售和管理数字资产的工具，包括用于发行的 Candy Machine 和用于交易的拍卖行。

### 工作原理

Metaplex 定义了 Solana 上 NFT 元数据的结构方式，包括名称、图片、特征和版税设置等属性。其 Candy Machine 程序自动化 NFT 集合发行，具有白名单、支付选项和揭示机制等功能。该协议还支持具有强制版税的可编程 NFT。

### 为什么重要

Metaplex 是 Solana NFT 生态系统的骨干。通过提供标准化基础设施，它确保了市场、钱包和工具之间的互操作性。其压缩 NFT 支持和可编程版税执行为更广泛的 NFT 行业树立了新标准。
