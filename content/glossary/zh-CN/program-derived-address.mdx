## 程序派生地址（PDA）

**程序派生地址**（PDA，Program Derived Address）是 Solana 上的一种特殊账户地址，由程序 ID 和一组种子确定性生成。与普通账户不同，PDA 没有对应的私钥，这意味着只有派生它们的程序才能代表它们签署交易。

### 工作原理

PDA 通过将程序 ID 与用户定义的种子（如用户的公钥或字符串）进行哈希来创建。生成的地址保证不落在椭圆曲线上，确保不存在对应的私钥。这允许程序以编程方式"拥有"账户并为其签名，支持托管账户和协议金库等模式。

### 为什么重要

PDA 是 Solana 程序架构的基础。它们使程序无需外部签名者即可控制账户，这对构建自主协议至关重要。它们还提供确定性寻址，允许任何人在给定相同种子的情况下派生出相同的账户地址。
