## Helius

**Helius** 是 Solana 区块链的顶级基础设施提供商，提供 RPC（远程过程调用）节点、Webhooks 和增强型 API，开发者使用这些工具在 Solana 上构建应用。可以把它理解为连接应用和 Solana 区块链的管道。

### 提供什么

Helius 提供高性能 RPC 端点，允许应用读写 Solana 区块链数据；增强型 API 用于获取解析的交易数据和数字资产信息；Webhooks 用于实时事件通知；以及 DAS（数字资产标准）API 用于 NFT 和代币元数据。这些工具为开发者节省了数月从零构建基础设施的时间。

### 为什么重要

就像网站需要网络托管提供商一样，区块链应用需要可靠的节点基础设施。Helius 是 Solana 生态系统中最受信赖的提供商之一，为许多热门应用提供支持。对用户来说，这意味着你日常使用的 Solana 应用可能在幕后运行在 Helius 基础设施上，让你的体验快速而可靠。
