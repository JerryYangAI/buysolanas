## Switchboard（交换板预言机）

**Switchboard** 是一个无许可的预言机网络，允许开发者将任何链下数据带到链上。它最初为 Solana 构建，支持可定制的数据源，能够从任何 API 获取信息，使其在 DeFi 和其他区块链应用中具有高度通用性。

### 工作原理

开发者通过指定要查询的数据源和聚合结果的方式来创建数据源。然后，预言机运营商网络获取并验证这些数据，将其发布到链上。Switchboard 还支持可验证随机函数（VRF），用于需要可证明公平随机性的应用，如游戏和 NFT 铸造。

### 为什么重要

Pyth 专注于金融价格数据，而 Switchboard 提供更广泛的预言机功能。任何通过 API 可用的数据都可以带到链上，支持体育博彩结果、天气数据和社交媒体指标等用例来驱动 Solana 上的智能合约。
